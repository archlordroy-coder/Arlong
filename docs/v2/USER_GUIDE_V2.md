# 📖 Guide Utilisateur Mboa Drive V2.0

## 🆕 Nouvelles Fonctionnalités Desktop

### 1. Système de Mise à Jour
Plus besoin de réinstaller ! L'application vérifie automatiquement les mises à jour au démarrage. Si une nouvelle version est disponible, un bandeau apparaît pour vous proposer le téléchargement.

### 2. WhatsApp Bot
Vous pouvez désormais envoyer vos documents archivés directement via WhatsApp.
- Rendez-vous dans les **Paramètres**.
- Scannez le code QR avec votre téléphone (WhatsApp > Appareils connectés).
- Une fois connecté, une option "Envoyer via WhatsApp" apparaîtra sur vos fichiers.

### 3. Assistant IA (Gemma 4)
Utilisez l'IA pour analyser vos documents.
- OCR : Extrayez le texte de vos images scannées.
- Description : Obtenez un résumé visuel de vos photos.
- Chat : Posez des questions sur vos archives à Arlong AI.

### 4. Partage de Dossiers
Collaborez avec d'autres utilisateurs en partageant vos dossiers ou espaces de travail avec des permissions spécifiques (Lecture, Édition, Admin).

---

## 🛠️ Interface Administration (Web)
Les administrateurs peuvent désormais gérer les versions de l'application via le portail web à l'adresse `/admin`.
Seules les versions marquées comme "Valides" seront proposées en mise à jour aux utilisateurs.
